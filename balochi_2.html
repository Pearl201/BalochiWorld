How to use

To type directly with the computer keyboard:

  • Type aa to get ā
  • To add a diacritical mark, type an apostrophe; example: t', j' (or c), H' (or x), d', s', S', T', G', k' (or g)
  • Type tt, dd, rr for the Balochi special characters
  • Type h' for ʰ
  • Type - for the hamza
  • Type y-- for ئ
  • Type &a, &u, &i to add the diacritics and && for the long consonant
